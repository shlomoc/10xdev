# **Antigravity**

## Overview

[Antigravity](https://antigravity.google) - Google's agent-first IDE and CLI.

Google's AI-powered development environment is built for agentic workflows across the editor, terminal, and browser.

## Gemini CLI Transition

Gemini CLI will be replaced by Antigravity CLI on June 18, 2026.

- [Gemini CLI GitHub](https://github.com/google-gemini/gemini-cli) - Existing Gemini CLI repository during the transition
- [Gemini CLI: Code & Create with an Open-Source Agent](https://learn.deeplearning.ai/courses/gemini-cli-code-and-create-with-an-open-source-agent) - DeepLearning.AI intro course for Gemini CLI
